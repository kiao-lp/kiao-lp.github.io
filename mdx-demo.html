Callout Examples

ℹ️ Note

This is an informational callout rendered as a JSX component at build time.

💡 Tip

MDX files compile to static HTML — no client-side JavaScript required.

⚠️ Migration

Only rename .md to .mdx when you need components. MDX is stricter than Pandoc markdown.

🔴 Breaking changes

HTML comments <!-- --> are not supported in MDX. Use {/* comments */} instead.
