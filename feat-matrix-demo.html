FeatMatrix

Compare options against prioritized features using MoSCoW requirements.

Set the priority for each feature, and watch options reorder by score.

How it works

  1. Set the priority (MoSCoW) for each feature using the dropdown in the Priority column
  2. Options are automatically scored and reordered — best match first
  3. Features are sorted by priority — Must-haves appear at the top

Scoring

Each feature contributes moscow_weight x support_score to an option's total:

MoSCoWWeightSupport LevelScore
Must8Out of the Box3
Should4Configuration2
Could2Customization1
Won't0Not Supported0

React usage

import { FeatMatrix } from 'feat-matrix';
import 'feat-matrix/styles.css';

<FeatMatrix features={features} options={options} />
